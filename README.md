

## What This Project Does
This project has two parts:
- `grade-evaluator.py` — reads a CSV of student grades, validates them, calculates the GPA, determines pass/fail status, and suggests which assignment to resubmit.
- `organizer.sh` — a Bash script that archives grades.csv with a timestamp and resets the workspace.
